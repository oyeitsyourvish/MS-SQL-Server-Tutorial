### &#x20;      **\* Git and GitHub \***



##### **# Basic Git Cmd**



1\) git init :-

&#x09;	- initiliaze repository.



2\) git status :-

&#x09;	- to check status.



3\) git add filename :-

&#x09;	- stage that exact file.



4\) git add . :-

&#x09;	- stage all files at a time.



5\) git commit -m "your comment" :-

&#x09;	- add comment and save stage.



6\) git log :-

&#x09;	- see the all previous comment with details.



7\) git log --oneline :-

&#x09;	- to see all comment in oneline.



8\) git config --global user.name "Your Name" :-

&#x09;	- it set the name of you git when you do work it use this name



9\) git config --global user.email "Your Email" :-

&#x09;	- it set the Email of you git when you do work it use this Email



10\) git restore --staged filename :-

&#x09;	- it you staged file using git add . but u dont want use this cmd.

&#x09;	- that specific file is unstage.





##### **# Single User Setup**



11\) .gitignore :- this is not cmd this the extession file which use to if we want to 		  not push any file from project to the GitHub but we want to stay 		  that file in project so in .gitignore file we add that file name to 		  ignore to push ex. doc.txt (file will be ignored) or build/ (folder 		  will be ignored).



12\) .gitkeep :- when we create an empty folder or directory so git cant be track it 		so if we want to track it then create .gitkeep file extension in that 		folder the that folder will be trackable.
